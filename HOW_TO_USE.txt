## 🚀 How to Get Your Website Live

### Step 1: Get Hosting
You need a web hosting provider to store your website files. Recommended South African hosts:
- **HostPapa** (hostpapa.co.za)
- **Hetzner** (hetzner.co.za)
- **Afrihost** (afrihost.com)

Cost: ~R30-100/month

### Step 2: Point Your Domain
Once you have hosting:
1. Log into your domain registrar
2. Go to DNS settings
3. Add an "A Record" pointing to your hosting server's IP address

### Step 3: Upload Files
1. Open your hosting control panel (cPanel)
2. Go to "File Manager"
3. Open the "public_html" folder
4. Upload ALL the files from this folder
5. Make sure files are inside public_html, not in a subfolder

### Step 4: Enable SSL
1. In cPanel, find "SSL/TLS"
2. Click "Install" on the free certificate
3. Your site will now work on https://

## ✏️ How to Update Your Website

### To update text on any page:
1. Open the HTML file (e.g., `index.html`)
2. Find the text you want to change
3. Edit it directly
4. Save the file
5. Re-upload to hosting

### To change product images:
1. Replace the image in the `images/` folder
2. Keep the same filename
3. Re-upload the image

### To add new products:
1. Open `offers.html`
2. Copy an existing product section
3. Paste it and change the image name and text
